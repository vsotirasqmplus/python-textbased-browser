Description

Every browser has a “back” button. If the user presses this button, the browser shows the previous web page. This feature can be realized using a stack. You save the pages visited by the user: google, wikipedia, bloomberg, but when the user types back, you will see the last visited pages in the reverse order: wikipedia, google.

Objectives

The functionality of the program is the same as in the previous task, but now the program has a new feature:

  1. The program should show the previous web page saved to a file if the user's input is the string back. Note that the last page you saved to a file is considered to be the current page, and when input is the string back, you should output the page that was before the current one. You can implement a stack to do this (but the current page shouldn't be put in the stack). For example, if the user inputs bloomberg.com, then nytimes.com, then back, after the command back the content of the web page bloomberg.com should be printed.
  2. If there are no more pages in the browser history, don’t output anything. For example, if the first input of the user is the string back, your program shouldn't output anything.

Example

The greater-than symbol followed by space (> ) represents the user input. Notice that it's not the part of the input.

> python browser.py dir-for-files
> bloomberg.com
