Imagine that you are a hacker and you've got access to a web server written in Python. The problem is that it asks for a password which you don't know. However, this server is open source so you know that there is a wrong_password() function which takes a password and checks it against real_password.

def wrong_password(password):
    return (password == "" or (not password and real_password)) or password != real_password

You can access this function. Write your code inside the solve() function: find the real_password and print it. Your program shouldn't read any input or call the function, just implement it.
